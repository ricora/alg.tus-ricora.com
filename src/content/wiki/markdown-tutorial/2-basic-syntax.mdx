## よく使う記法

### 見出し

```md
# h1

## h2

### h3
```

### 箇条書き

```md
- item 1
- item 2
```

### 強調とインラインコード

```md
これは**強調**です。
これは`code`です。
```

### リンクと画像

```md
[RICORA](https://alg.tus-ricora.com/)
![logo](https://avatars.githubusercontent.com/u/33452053)
```

### コードブロック

````md
```ts
console.log("Hello")
```
````

### 表

```md
| 名前     | 説明             |
| -------- | ---------------- |
| Markdown | 軽量マークアップ |
```

### 数式

```md
$ x^2 + y^2 = 1 $
```

## 記法よりも大事なこと

記法を全部覚えるより、文書の構造をはっきりさせることが重要です。

- 見出しは階層を飛ばさない
- 1段落1トピックを意識する
- 例と結論を近い位置に置く
