## CLIの基本

CLIは、文字入力でコンピュータを操作する方法です。
最初は難しく見えますが、同じ操作を再現しやすいという強みがあります。

## コマンドの形

```sh
(コマンド) (引数) (オプション)
```

- 引数: どの対象に対して実行するか
- オプション: 動作を調整する指定

## 最初に覚えるコマンド

### 現在地を確認

```sh
pwd
```

### 移動

```sh
cd path/to/dir
cd ..
cd ~
```

### 一覧表示

```sh
ls
ls -la
```

### 作成

```sh
mkdir new-dir
touch note.txt
```

### 削除

```sh
rm note.txt
```

## 安全に使うためのポイント

- 危険な操作の前に pwd と ls で現在地を確認する
- 削除系コマンドは対象を必ず再確認する
- 不明なコマンドはヘルプやマニュアルを参照する

```sh
command --help
man command
```
